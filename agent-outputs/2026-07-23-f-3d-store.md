# F-batch: the storefront becomes a free-standing 3D store

Sixth session on `claude/3d-models-led-text-lrcgbn`, addressing F1.

The single-façade storefront is now a full corner-shop building
(5.455 × 3.818 × 4.0 units) with four real elevations and a capped
flat roof:

- **Front (+Z)**: the existing shopfront unchanged — fascia sign
  (`children`), window poster, glazed door, cornice, pilasters.
- **Windowed side (+X)**: the same composition without the door —
  riser, pilaster-to-pilaster glazing with mullions and transom lights,
  cornice, brick band — carrying its own live fascia sign (`sideSign`).
- **Brick side (−X) and rear (−Z)**: full-height running-bond brickwork
  (the canvas-baked texture generalized to arbitrary panels), each with
  an optional painted-wall mural (`sideWall`, `rearWall`) — the
  hand-painted-ad look.
- **Roof**: a dark membrane cap overhanging the parapet line; not
  mockup-able, registered as a screen occluder alongside the body.

All four elevations are live DOM. Docs: props table + copy rewritten,
the chroma map now labels all five surfaces on an auto-rotating
building, a new walk-around example, a `StoreMuralArt` demo art, and a
`device=store` harness mode.

Verified with a six-shot orbit (front, corner, windowed side, brick
side, rear, top): `images/2026-07-23-f-store-*.png`.
