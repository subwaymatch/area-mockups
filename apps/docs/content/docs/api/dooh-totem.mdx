`<DOOHTotem>` and `<DOOHTotemMockup>` render a digital out-of-home street
totem (digital 6-sheet class): a rounded street-furniture enclosure on a low
plinth with dark glass on both faces, and a live portrait 540×960 display —
the digital sibling of the [billboard](/docs/api/billboard) and
[bus shelter](/docs/api/bus-shelter).

<ObjectDemo demo="totem-basic" checker />

```tsx
<DOOHTotemMockup deviceProps={{ rotation: [0, -0.18, 0] }}>
  <YourCreative />
</DOOHTotemMockup>
```

## Props

Beyond the [shared screen props](/docs/screen-content):

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `back` | `ReactNode` | - | Creative on the back (−Z) display — real totems are double-sided. |
| `color` | `string` | `'#2f333a'` | Enclosure colorway. |
| `resolution` | `number` | `540` | Virtual display width; 540 gives 540×960 (a 1080×1920 grid at 2x). |
