title: Product box
description: A retail carton with all six panels as live print areas.

`<ProductBox>` and `<ProductBoxMockup>` render a standing retail carton
(190×265×55 mm — the classic cereal/software hero proportion) in satin
coated stock with crisp fold edges and the tuck-flap seam on top. All six
panels are live DOM — the front (bare children) plus the `Right`, `Left`,
`Top`, `Bottom` and `Back` slots — so a standard 3/4 pose shows three
printed faces at once, all
sharing one dpi, and no pose reveals bare stock unless you want it to.
Pass `size` (in millimeters) for a carton of any other dimensions.

Front, side and top, each its own live surface.

<ObjectDemo demo="box-basic" checker />

```tsx
<ProductBoxMockup rotation={[0, -0.5, 0]}>
  <FrontPanel />
  <ProductBoxMockup.Right><SidePanel /></ProductBoxMockup.Right>
  <ProductBoxMockup.Top><TopPanel /></ProductBoxMockup.Top>
</ProductBoxMockup>
```

## Props

Beyond the [shared screen props](/docs/screen-content):

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `size` | `{ width, height, depth }` | `190×265×55` | Real size in **millimeters**. The longest edge normalizes to the stage, so any size fills the camera at true proportions. |
| `color` | `string` | `'#f4f1ea'` | Carton stock: unprinted panels and fold edges. |
| `surfaceBackground` | `string` | `'#ffffff'` | CSS background behind each printed panel. |
| `resolution` | `number` | `480` | Virtual front-panel width in CSS px; other panels share its dpi. |

## Regions

| Slot | Panel |
| --- | --- |
| `<ProductBoxMockup.Front>` (bare children) | Front panel. |
| `<ProductBoxMockup.Right>` | Right side panel (the one visible in a 3/4 pose). |
| `<ProductBoxMockup.Left>` | Left side panel. |
| `<ProductBoxMockup.Top>` | Top panel, reading from the front. |
| `<ProductBoxMockup.Bottom>` | Bottom panel, reading from the front when tipped. |
| `<ProductBoxMockup.Back>` | Back panel. |

Every slot takes per-surface overrides — `background`, `resolution`,
`interactive`, `dragToRotate`, `style` — over the mockup-level defaults.

## More examples

### Every panel printed

Front, back, both sides, top and bottom — brew steps and barcode on the
back, the roast-date stamp underneath. Drag it anywhere; it stays designed.

<ObjectDemo demo="box-full" />

  <ProductBoxMockup.Top><TopPanel /></ProductBoxMockup.Top>
