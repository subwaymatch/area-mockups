`<Billboard>` and `<BillboardMockup>` render a classic 14'×48' highway
bulletin: steel face panel with trim lip, the torsion box and slatted apron
skirt the monopole actually connects to, a maintenance catwalk with a
two-rail guardrail, gooseneck floodlights along the top edge, and a
foundation collar at grade. The face is a live 1200×350 DOM surface. The
mockup grounds the pole on a virtual ground plane.

<ObjectDemo demo="billboard-basic" checker />

```tsx
<BillboardMockup rotation={[0, -0.18, 0]}>
  <YourCreative />
</BillboardMockup>
```

## Props

Beyond the [shared screen props](/docs/screen-content):

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `color` | `string` | `'#2c313a'` | Steel color: panel, apron, pole, catwalk, and fixtures. |
| `surfaceBackground` | `string` | `'#ffffff'` | CSS background behind your face content. |
| `resolution` | `number` | `1200` | Virtual face width in CSS px; height follows the 14:48 bulletin. |

The face is very wide (3.4:1) — design the creative like an actual bulletin:
one message, big type.

## Regions

The face is the only region — bare children fill it. Wrap the creative in
`<BillboardMockup.Face>` (or `<Billboard.Face>`) only to set per-surface
overrides: `background`, `resolution`, `interactive`, `dragToRotate`, `style`.

## Mockup-able areas

Every live surface filled with chroma green, labeled with the slot that
feeds it — the fastest way to see exactly where your content lands and
what stays 3D hardware.

## More examples

### A roadside pair

Bare `<Billboard>` objects in one [`MockupCanvas`](/docs/api/mockup-canvas),
staggered in depth like successive highway boards.

<ObjectDemo demo="billboard-pair" />

```tsx
<MockupCanvas camera={{ position: [0, 0.4, 13.4], fov: 40 }} shadowY={-1.5}>
  <group position={[0, 0.6, 0]}>
    <Billboard position={[-3.4, 0, -2.4]} rotation={[0, 0.5, 0]} color="#3a3f49">
      <TeaserCreative />
    </Billboard>
    <Billboard position={[1.6, 0, 0]} rotation={[0, -0.12, 0]}>
      <MainCreative />
    </Billboard>
  </group>
</MockupCanvas>
```
