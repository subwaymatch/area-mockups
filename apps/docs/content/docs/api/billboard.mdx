`<Billboard>` and `<BillboardMockup>` render a classic 14'×48' highway
bulletin: steel face panel on a monopole with a foundation collar, a
maintenance catwalk with railing, and gooseneck floodlights along the top
edge. The face is a live 1200×350 DOM surface. The mockup grounds the pole on
a virtual ground plane.

```tsx
<BillboardMockup deviceProps={{ rotation: [0, -0.2, 0] }}>
  <YourCreative />
</BillboardMockup>
```

## Props

Beyond the [shared screen props](/docs/screen-content):

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `color` | `string` | `'#2c313a'` | Steel color: panel, pole, catwalk, and fixtures. |
| `faceBackground` | `string` | `'#ffffff'` | CSS background behind your face content. |
| `resolution` | `number` | `1200` | Virtual face width in CSS px; height follows the 14:48 bulletin. |

The face is very wide (3.4:1) — design the creative like an actual bulletin:
one message, big type.
