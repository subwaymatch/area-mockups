title: Storefront
description: A free-standing corner shop with live surfaces on all four elevations.
---

`<Storefront>` and `<StorefrontMockup>` render a free-standing single-unit
corner shop with four real elevations and a capped flat roof. The front is
the classic high-street shopfront — painted timber surround with cornice
and pilasters, a 600 mm stall riser, big display glazing with a mullion, a
glazed door on the right. The windowed side repeats the same composition
without the door and carries its own fascia sign; the other side and the
rear are running-bond brickwork. Every elevation is mockup-able: the two
fascia signs, the window poster, and painted-wall murals on the two brick
elevations (the roof stays hardware).

<ObjectDemo demo="storefront-basic" checker />

```tsx
<StorefrontMockup windowPoster={<YourPoster />} deviceProps={{ rotation: [0, -0.25, 0] }}>
  <YourSign />
</StorefrontMockup>
```

## Props

Beyond the [shared screen props](/docs/screen-content):

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `windowPoster` | `ReactNode` | - | Poster displayed inside the front's left window bay. |
| `sideSign` | `ReactNode` | - | Fascia sign on the windowed side (+X) — the door-less twin elevation. |
| `sideWall` | `ReactNode` | - | Painted-wall mural on the brick side (−X). Plain brick when omitted. |
| `rearWall` | `ReactNode` | - | Painted-wall mural on the rear (−Z). Plain brick when omitted. |
| `color` | `string` | `'#2e4638'` | Shopfront paint: fascia surrounds, frames, door, stall risers. |
| `wallColor` | `string` | `'#8d7a6a'` | Masonry on every elevation. |
| `faceBackground` | `string` | `'#ffffff'` | CSS background behind sign, poster and mural content. |
| `resolution` | `number` | `800` | Virtual fascia sign width in CSS px. The window poster is a fixed 420 px wide; murals are 700 px. |

The pavement sits `STOREFRONT.standHeight` below the origin — the mockup
grounds the shadow there automatically.
