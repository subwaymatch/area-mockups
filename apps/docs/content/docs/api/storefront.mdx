---
title: Storefront
description: A free-standing corner shop with live surfaces on all four elevations.
---

`<Storefront>` and `<StorefrontMockup>` render a free-standing single-story
corner shop: four glazed shopfront elevations under a capped flat roof,
the parapet sitting just above the cornice so the painted composition IS
the whole building. The front is the classic high-street shopfront —
painted timber surround with cornice and pilasters, a 600 mm stall riser,
big display glazing with a mullion, a glazed door on the right. The other
three elevations repeat the same composition without the door — windows
only — and each carries its own live fascia sign. Every elevation is
mockup-able (the roof stays hardware).

<ObjectDemo demo="storefront-basic" checker />

```tsx
<StorefrontMockup windowPoster={<YourPoster />} deviceProps={{ rotation: [0, -0.25, 0] }}>
  <YourSign />
</StorefrontMockup>
```

## Props

Beyond the [shared screen props](/docs/screen-content):

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `windowPoster` | `ReactNode` | - | Poster displayed inside the front's left window bay. |
| `leftSign` | `ReactNode` | - | Fascia sign on the left elevation (+X as you face the shop). |
| `rightSign` | `ReactNode` | - | Fascia sign on the right elevation (−X as you face the shop). |
| `rearSign` | `ReactNode` | - | Fascia sign on the rear elevation (−Z). |
| `color` | `string` | `'#2e4638'` | Shopfront paint: fascia surrounds, frames, door, stall risers. |
| `faceBackground` | `string` | `'#ffffff'` | CSS background behind sign and poster content. |
| `resolution` | `number` | `800` | Virtual fascia sign width in CSS px. The window poster is a fixed 420 px wide. |

The pavement sits `STOREFRONT.standHeight` below the origin — the mockup
grounds the shadow there automatically.
