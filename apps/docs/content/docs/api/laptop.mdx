`<Laptop>` and `<LaptopMockup>` render a MacBook Air 13" (M5) style laptop:
unibody deck with the real 78-key Magic Keyboard layout (printed legends,
rendered as a single instanced draw call), trackpad, hinge band, ports, and a
notched display.

```tsx
<LaptopMockup color="#aec6d9" deviceProps={{ rotation: [0, -0.35, 0] }}>
  <YourDesktopApp />
</LaptopMockup>
```

## Props

Beyond the [shared screen props](/docs/screen-content):

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `color` | `string` | `'#e3e4e6'` | Aluminum finish. Try Sky Blue `#aec6d9`, Starlight `#e8e0d4`, Midnight `#2e3642`. |
| `notch` | `boolean` | `true` | Camera notch overlay at the top of the display. |
| `openAngle` | `number` | `110` | Lid angle in degrees between deck and screen (90 is upright). |
| `resolution` | `number` | `1280` | Virtual display width; 1280 gives 1280×832, exactly the MacBook Air default scaled resolution, so desktop breakpoints apply. |

There is no `orientation` prop; laptops only have one pose.
