`<VinylRecord>` and `<VinylRecordMockup>` render a 12" LP: a 313 mm square
hollow jacket with live front (and optional back) cover art, and a glossy
302 mm disc, grooved on both sides, sliding out of the sleeve far enough to
clear the whole label — concentric track-gap sheen rings, dead-wax annulus,
spindle hole, and the 100 mm center label as a live **circular** DOM area.

<ObjectDemo demo="vinyl-basic" checker />

```tsx
<VinylRecordMockup label={<Label />} deviceProps={{ rotation: [0, -0.2, 0] }}>
  <CoverArt />
</VinylRecordMockup>
```

## Props

Beyond the [shared screen props](/docs/screen-content):

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `back` | `ReactNode` | - | Back cover design. |
| `label` | `ReactNode` | - | Side-A center label — a live circle, like the round watch face. |
| `backLabel` | `ReactNode` | - | Side-B center label on the disc's other face. |
| `vinylColor` | `string` | `'#0b0b0d'` | Disc color. Classic black; try pressing colors. |
| `color` | `string` | `'#f2efe8'` | Jacket stock (edges and unprinted faces). |
| `faceBackground` | `string` | `'#ffffff'` | CSS background behind cover, back and label. |
| `resolution` | `number` | `520` | Virtual cover width in CSS px; the label shares its dpi. |

## More examples

### Back cover and side B

`back` prints the tracklist side of the jacket and `backLabel` the disc's
side-B label, so the record reads right from behind too.

<ObjectDemo demo="vinyl-full" />

```tsx
<VinylRecordMockup autoRotate back={<Tracklist />} label={<SideA />} backLabel={<SideB />}>
  <CoverArt />
</VinylRecordMockup>
```
