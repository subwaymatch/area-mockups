---
title: Greeting card
description: A standing A7 card with four live faces — covers and inside spread.
---

`<GreetingCard>` and `<GreetingCardMockup>` render a folded 5"×7" (A7) card
standing like a tent, fold toward the viewer: `children` is the front cover,
`backCover` the other outward face, and `insideLeft`/`insideRight` print the
spread you see from behind. Each face gently shades into the crease, like
paper actually bowing.

## Props

Beyond the [shared screen props](/docs/screen-content):

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `backCover` | `ReactNode` | - | The left outward face. |
| `insideLeft`, `insideRight` | `ReactNode` | - | The inside spread. |
| `openAngle` | `number` | `65` | Angle between the panels (display sweet spot is 60–75). |
| `color` | `string` | `'#f6f3ec'` | Card stock: edges and unprinted faces. |
| `faceBackground` | `string` | `'#ffffff'` | CSS background behind each printed face. |
| `resolution` | `number` | `380` | Virtual panel width in CSS px; height follows the A7 panel. |
