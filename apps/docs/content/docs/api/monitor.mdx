`<Monitor>` and `<MonitorMockup>` render an Apple Studio Display style 27"
monitor (2026 generation), proportioned from the product photography:
all-aluminum enclosure with an edge-to-edge glass front (uniform black
bezel, hairline aluminum rim), the barely-there centered camera, the
generation's gloss-black Apple mark on the back, rear Thunderbolt and USB-C
port row, speaker perforations along the bottom edge, and the signature
tilt stand — a wide thin-sheet arm hanging from its hinge (machined pivot
caps on the sides) with the circular cable-routing hole punched clean
through, straddling the enclosure's bottom edge — plus (faithfully) no
power button. The mockup grounds the stand on a virtual desk plane.

```tsx
<MonitorMockup deviceProps={{ rotation: [0, -0.25, 0] }}>
  <YourDesktopApp />
</MonitorMockup>
```

## Props

Beyond the [shared screen props](/docs/screen-content):

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `colorway` | `string` | – | Retail colorway id from `MONITOR_COLORWAYS` (`silver`). Explicit `color` overrides it. |
| `color` | `string` | `'#c8cbd0'` | Aluminum colorway (enclosure and stand). |
| `resolution` | `number` | `2560` | Virtual display width; 2560 gives 2560×1440, the Studio Display's logical desktop, so real desktop layouts apply. |

There is no `orientation` prop; it lives on a desk.
