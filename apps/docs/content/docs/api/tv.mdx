---

`<TVSet>` and `<TVSetMockup>` render a modern 65" flat-screen: a
near-bezel-less 16:9 panel with a subtle off-state gloss, thin edges with a
shallow electronics bulge low on the back, and two splayed blade feet. The
screen is a live 1920×1080 DOM surface — full desktop layouts, showreels,
and `<video>` all apply.

<ObjectDemo demo="tv-basic" checker />

```tsx
<TVSetMockup deviceProps={{ rotation: [0, -0.22, 0] }}>
  <YourShowreel />
</TVSetMockup>
```

## Props

Beyond the [shared screen props](/docs/screen-content):

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `color` | `string` | `'#15171b'` | Enclosure colorway (frame, back, feet). |
| `resolution` | `number` | `1920` | Virtual display width; 1920 gives 1920×1080. |

There is no `orientation` prop; it lives on a media stand.
