title: Bus shelter
description: A glass transit shelter with a backlit live 6-sheet lightbox.
---

`<BusShelter>` and `<BusShelterMockup>` render a JCDecaux-class glass transit
shelter: a flat roof slab cantilevered off two slim back posts, full-width
glass back wall, floating bench — and the 6-sheet lightbox (1185×1750 mm)
standing as the structural end panel, with backlit diffusers behind a live
480×709 poster on each face.

The arrivals display speaks LED natively: give `<BusShelter.Arrivals>` an
array of strings and each becomes a glowing dot-matrix board row (rows scroll
when they overflow), a single string for a one-line message — or any React
node for a fully custom board.

<ObjectDemo demo="shelter-basic" checker />

```tsx
<BusShelterMockup rotation={[0, -0.55, 0]}>
  <YourPoster />
  <BusShelterMockup.Arrivals>
    {['12  Harbor Loop      2 min', '4X  Union Sq Express  7 min', '31  Larchmont        12 min']}
  </BusShelterMockup.Arrivals>
</BusShelterMockup>
```

## Props

Beyond the [shared screen props](/docs/screen-content):

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `color` | `string` | `'#2f333a'` | Street-furniture steel: roof, posts, frames, bench supports. |
| `surfaceBackground` | `string` | `'#ffffff'` | CSS background behind the poster (backlit white). |
| `resolution` | `number` | `480` | Virtual poster width in CSS px; height follows the 6-sheet. |

The lightbox faces ±X (along the sidewalk) — pose the shelter in
three-quarter view to feature it.

## Regions

| Slot | Surface |
| --- | --- |
| `<BusShelterMockup.Poster>` | Outward 6-sheet lightbox face (bare children). |
| `<BusShelterMockup.Inner>` | The inward (waiting-area) lightbox face. |
| `<BusShelterMockup.Arrivals>` | Live RTPI arrivals display hanging under the roof. Strings get the built-in dot-matrix LED board (one row per string); a node renders as-is. |
| `<BusShelterMockup.ArrivalsBack>` | The waiting-area face of the arrivals board. Mirrors `Arrivals` unless given its own content. |

Every slot takes per-surface overrides — `background`, `resolution`,
`interactive`, `dragToRotate`, `style` — over the mockup-level defaults. The
same slots exist on the bare `<BusShelter>`.
