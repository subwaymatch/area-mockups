`<Phone>` renders a Galaxy S25 family device; `<PhoneMockup>` is the
all-in-one version. All four variants keep true relative sizes and per-model
camera architecture.

<ObjectDemo demo="phone-basic" checker />

```tsx
<PhoneMockup color="#15171d" frameColor="#4d5260" deviceProps={{ rotation: [0, -0.25, 0] }}>
  <TapCounter />
</PhoneMockup>
```

## Props

Beyond the [shared screen props](/docs/screen-content) and, on the mockup, the
[shared mockup props](/docs/api):

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `variant` | `'s25' \| 's25plus' \| 's25ultra' \| 's25edge'` | `'s25'` | Which Galaxy S25 family device: true relative sizes, per-model cameras (Ultra five-element array, Edge pill island), and logical resolutions. |
| `orientation` | `'portrait' \| 'landscape'` | `'portrait'` | Landscape lays the device on its side and swaps the virtual display (for example 780×360 on the S25) with upright content. |
| `color` | `string` | `'#101216'` | Back panel colorway. |
| `frameColor` | `string` | `'#4a4f59'` | Metal frame, buttons, and camera rings. |
| `punchHole` | `boolean` | `true` | Front camera punch hole overlay. |
| `resolution` | `number` | per variant | CSS pixel width of the virtual display; height follows the panel aspect (360 gives 360×780 on the S25). |

## Colorways

Try Icy Blue `#c9d6e8`, Navy `#2c3344`, Mint `#c8e6d2`, or Marble Gray
`#d3d6dd` with `frameColor="#b6bac4"`.

## More examples

Every example is live: drag to spin it, and the screens really work.

### Colorways with float and auto-orbit

<ObjectDemo demo="phone-lock" />

```tsx
<PhoneMockup autoRotate float color="#d3d6dd" frameColor="#b6bac4" screenBackground="#000">
  <LockScreen />
</PhoneMockup>
```

### Landscape, at the Ultra's real resolution

`orientation="landscape"` swaps the virtual display to 832×384 on the
S25 Ultra — your responsive layout reacts exactly like on the device.

<ObjectDemo demo="phone-landscape" />

```tsx
<PhoneMockup variant="s25ultra" orientation="landscape" color="#2e3238" frameColor="#565b64">
  <MusicPlayer />
</PhoneMockup>
```
