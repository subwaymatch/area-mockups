`<RollupBanner>` and `<RollupBannerMockup>` render the trade-show standard
850×2000 mm retractable banner: an aluminum cassette base with rounded ends,
end caps and swivel feet, the graphic rising out of the cassette slot to a
slim top clamp rail, with the support pole hidden behind. The graphic is a
live 420×988 DOM surface. The mockup grounds the cassette on a virtual
floor.

<ObjectDemo demo="banner-basic" checker />

```tsx
<RollupBannerMockup deviceProps={{ rotation: [0, 0.18, 0] }}>
  <YourBannerArt />
</RollupBannerMockup>
```

## Props

Beyond the [shared screen props](/docs/screen-content):

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `size` | `{ width?, height? }` | `850×2000` | Physical graphic in mm — e.g. `{ width: 1000 }` for the wide stand. The cassette resizes with it. |
| `color` | `string` | `'#b9bdc4'` | Hardware color: cassette, feet, pole, top rail. |
| `graphicBackground` | `string` | `'#ffffff'` | CSS background behind your graphic content. |
| `resolution` | `number` | `420` | Virtual graphic width in CSS px; height follows the 850×2000 sheet. |
