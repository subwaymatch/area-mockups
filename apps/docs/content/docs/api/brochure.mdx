---
title: Brochure (tri-fold)
description: A standing letter Z-fold with three live panels, front and back.
---

`<Brochure>` and `<BrochureMockup>` render a US letter tri-fold standing in a
zig-zag (Z-fold) accordion. Each of the three 93×216 mm panels is its own
live 360×835 DOM surface — and the reverse faces can be printed too. Panels
shade gently toward the folds, like paper actually bowing.

## Props

Beyond the [shared screen props](/docs/screen-content):

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `panels` | `[ReactNode?, ReactNode?, ReactNode?]` | - | Content per panel, left to right. Wins over `children` where provided. |
| `backPanels` | `[ReactNode?, ReactNode?, ReactNode?]` | - | Reverse-side content, left to right as seen from the back. |
| `foldAngle` | `number` | `24` | Zig-zag fold angle in degrees. `0` lays the sheet out flat. |
| `paperColor` | `string` | `'#f5f4f0'` | Stock color of unprinted faces and paper edges. |
| `panelBackground` | `string` | `'#ffffff'` | CSS background behind each panel's content. |
| `resolution` | `number` | `360` | Virtual width of one panel in CSS px; height follows the panel aspect. |

The fold keeps every panel centered on the group origin, so the brochure
orbits about its own visual center at any `foldAngle`.

## Examples

### All three panels

Pass an array — left to right. Each panel is independent live DOM.

<ObjectDemo demo="brochure-panels" />

```tsx
<BrochureMockup panels={[<Front />, <Middle />, <Back />]} />
```

### Flat, before folding

`foldAngle={0}` lays the sheet out flat — useful for reviewing the design as
the full 11" spread it prints as.

<ObjectDemo demo="brochure-flat" />

```tsx
<BrochureMockup foldAngle={0} panels={[<Front />, <Middle />, <Back />]} />
```

### One designed panel on bare stock

`children` is shorthand for the first panel; the rest show `paperColor`
stock — a work-in-progress state that reads clearly in reviews.

<ObjectDemo demo="brochure-single" />

```tsx
<BrochureMockup paperColor="#e9e4d8" float>
  <Front />
</BrochureMockup>
```
